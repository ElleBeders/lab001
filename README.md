# Задание 1

## Задача 1

### Текст задачи

Сформировать список из чисел, на 1 меньших, чем вводимые значения.

### Алгоритм решения

Пользователь вводит числа через энтер, которые записываются в список с помощью нисходящей рекурсии. Стоп-сигналом для ввода является значение 000, тогда программа получает готовый список и уменьшает на 1 все числа нём, после чего выводи его на экран.

# Задание 2

## Задача 1

### Текст задачи

Сколько строк введено? (Признак конца ввода — пустая строка) 

### Алгоритм решения

Погружаемся в рекурсию, пока не будет введена пустая строка. Когда вводиться пустая строка, нерекурсивная ветвь возращает 0, и происходит подсчёт погружений в рукурсию (ввод непустых строк).

# Задание 3

## Задача 1

### Текст задачи

Создайте собственные функции для выполнения основных операций над списками (добавление/удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).

### Алгоритм решения

Запрашиваем у пользователя выбор действия с помощью функции меню. На основе выбора переходим к слдующей функции (добавить элемент / удалить / найти / соединить списки / найти элемент по номеру).
Функция "добавить" запрашивает список и элемент, затем вносит в выбранный список элемент путём соединения списка из данного элемента и уже существующего. Если выбранный список пуст (ещё не существует), функция его создст.
Функция "удалить" запрашивает список и элемент, затем поэлементно перебирает список, складывая элементы во временный список, пока не дойдёт до нужного. Затем функция соединяет временный список и остаток от изначального списка, упуская заданный элемент.
